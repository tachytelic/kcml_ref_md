CONTINUE LOAD <span style="font-size: 16pt ; ">command</span>

------------------------------------------------------------------------

General Form:\
\
     CONTINUE LOAD\
\

------------------------------------------------------------------------

The CONTINUE LOAD command will restart a program that has been stopped either with a [STOP](STOP.htm) or a [TRAP](TRAP.htm) statement or has been interrupted by a Ctrl+BREAK, and stop it again immediately before the first statement to be executed after a new program has been loaded in a [LOAD](LOAD.htm) statement.

The CONTINUE LOAD command can be executed directly within the Workbench either as a single keystroke or from the toolbar.
