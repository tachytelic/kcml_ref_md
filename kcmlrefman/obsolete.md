Obsolete statements

------------------------------------------------------------------------

The following statements were supported in previous versions of KCML but have now been dropped.

| Statement | Purpose |
|----|----|
| \$REAL | This was used in DOS KCML up to version 4.0 to force variables to be allocated in the first 640kb of real memory to permit exchange of data with external C routines. It has no effect on Unix or Windows version of KCML. |
