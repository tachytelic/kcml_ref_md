Related products

KCML client

The KCML Client Kclient is a freely distributable client application that runs on Windows 9x or Windows NT and displays forms or text windows under the control of a KCML server process.

Kerridge database

KCML has a built in relational database which uses fast B Tree indexing. It supports access to tables either by SQL or by direct rowset (ISAM) operations.

Kerridge Mail

Kerridge Mail, or KMail for short, is a Windows based mail program for PC's connected to a UNIX host via the TCP/IP protocol. KMail utilises industry standard mail protocols and format to ensure that mail messages can be exchanged between different mail systems. A UNIX host is used as a router and a storage area for messages sent, KMail connects to the host in order to send and retrieve mail.

Kprint

Kprint is a distributed printing concept ideally suited for producing rich printed output on laser printers separated from the KCML server by a Wide Area Network. It allows plain paper to be substituted for expensive multipart preprinted stationery and no changes to the applications are necessary. The basic text of a document can be sent as ASCII text using the Internet Standard LPD protocol to a Kprint server running on a NT or Windows 95 PC. The Kprint server then combines the text with a document definition and generates a rich document which may use bitmaps, multiple fonts and line drawing. Document definitions are distributed automatically and cached in the remote servers. Kprint also has fax and email support and can print standard reports too. There is a easy to use WYSIWYG editor to generate form templates.
