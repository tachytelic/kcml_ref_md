Kclient

KTERM=Kclient

This is the native terminal for KCML running in client server mode. It runs under Microsoft Windows 9x, NT and CE. As well as supporting GUI forms it also has a text mode capability emulating both the VT220 and the native text mode KCML terminal. It is the reference implementation for a KCML terminal and therefore has full keyboard and screen support.

It is fully international and uses the native code page of the version of Windows it runs on. This means there is no support for underlining by ORing with HEX(80).

See the [Kclient chapter](mk:@MSITStore:kclient.chm::/introduction2.htm) of this manual for more about configuring and using Kclient.
