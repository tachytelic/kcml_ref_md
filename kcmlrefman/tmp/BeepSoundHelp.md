Beep sounds for form .Beep() method

These constants define the type of sound the [.Beep()](mk:@MSITStore:kcmlforms.chm::/tmp/PROP_DLG_BEEP.htm) method generates.

| Constant name       | Value | Description    |
|---------------------|-------|----------------|
| \_BEEP_NONE         | 0     | No sound       |
| \_BEEP_CRITICALSTOP | 1     | Critical error |
| \_BEEP_QUESTION     | 2     | Question       |
| \_BEEP_EXCLAMATION  | 3     | Exclamation    |
| \_BEEP_ASTERISK     | 4     | Asterisk       |
| \_BEEP_DEFAULT      | 5     | Default        |
