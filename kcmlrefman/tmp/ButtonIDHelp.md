Button IDs

The [MessageBox()](../MessageBox.htm) Windows API will return the ID of the button that was pressed. See also [Flags for MessageBox()](MessageBoxHelp.htm)

| Constant name | Value | Description |
|---------------|-------|-------------|
| \_IDOK        | 1     | OK          |
| \_IDCANCEL    | 2     | Cancel      |
| \_IDABORT     | 3     | Abort       |
| \_IDRETRY     | 4     | Retry       |
| \_IDIGNORE    | 5     | Ignore      |
| \_IDYES       | 6     | Yes         |
| \_IDNO        | 7     | No          |
| \_IDCLOSE     | 8     | Close       |
| \_IDHELP      | 9     | Help        |
