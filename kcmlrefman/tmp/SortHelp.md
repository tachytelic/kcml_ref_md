KCML Constants

SORT key descriptor constants

These constants control the behaviour [SORT](../SORT.htm) statement.

| Constant name          | Value | Description                   |
|------------------------|-------|-------------------------------|
| \_SORT_DESCENDING      | 0x20  | Sort descending               |
| \_SORT_COLLATESEQ      | 0x40  | Sort using collating sequence |
| \_SORT_CASEINSENSITIVE | 0x80  | Case insensitive sort         |
