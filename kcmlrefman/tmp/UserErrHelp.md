KCML Constants

Return values for programs executed by kmake

| Constant name | Value | Description |
|----|----|----|
| \_KMAKE_RETURN_SUCCESS | 0 | Successful |
| \_KMAKE_RETURN_WARNING | 1 | Code generated, but with warnings |
| \_KMAKE_RETURN_ERROR | 2 | No code generated due to errors |
| \_KMAKE_RETURN_FATAL | 3 | No code generated due to fatal errors such as i/o errors |
