KCML Constants

WRITE LOG level codes

These constants determine the severity of a message written to the system log when using the [WRITE LOG](../WRITE_LOG.htm) statement.

| Constant name     | Value | Description |
|-------------------|-------|-------------|
| \_LOG_INFORMATION | 0     | Information |
| \_LOG_WARNING     | 1     | Warning     |
| \_LOG_ERROR       | 2     | Error       |
