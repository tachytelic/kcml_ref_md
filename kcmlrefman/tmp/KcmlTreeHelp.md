KCML Constants

Tree node types for Tree Add() method

These constants define the type of a Tree node when using the [.Add()](mk:@MSITStore:kcmlforms.chm::/tmp/PROP_TREEITEM_ADD.htm) method.

| Constant name        | Value | Description                          |
|----------------------|-------|--------------------------------------|
| \_TREE_UNSORTEDCHILD | 0x0   | Default basic node                   |
| \_TREE_PARENT        | 0x1   | Node has children                    |
| \_TREE_SORT          | 0x2   | Sort forward                         |
| \_TREE_SORTREVERSE   | 0x4   | Sort backwards                       |
| \_TREE_FOLDERSFIRST  | 0x8   | Items with children are listed first |
