KDB constants

These are constants commonly used with the [Kerridge DataBase](mk:@MSITStore:kdb.chm::/) routines.

| Constant name            | Value | Description                              |
|--------------------------|-------|------------------------------------------|
| \_KDB_MAX_TABLENAME_LEN  | 35    | Maximum table name length                |
| \_KDB_MAX_KEY_LEN        | 160   | Maximum key length                       |
| \_KDB_MAX_INDEX          | 18    | Maximum number of indices for a table    |
| \_KDB_MAX_KEY_COLS       | 16    | Maximum number of columns per index      |
| \_KDB_MAX_WS_INDEX       | 4     | Maximum number of WS indices for a table |
| \_KDB_MAX_WS_KEY_COLS    | 8     | Maximum number of WS columns per index   |
| \_KDB_MAX_ROWID_LEN      | 18    | Row ID length                            |
| \_KDB_MAX_HANDLES        | 1024  | Maximum number of handles                |
| \_KDB_MAX_CONNECT        | 8     | Maximum number of connections            |
| \_KDB_AUTO_HANDLE        | 0     | Autoallocate a handle                    |
| \_KDB_TEMP_HANDLE        | -1    | Autoallocate a temporary handle          |
| \_KDB_DEFAULT_CONNECTION | 1     | Default legacy connection handle         |
