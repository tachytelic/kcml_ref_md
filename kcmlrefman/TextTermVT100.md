Generic DEC VT100

KTERM=vt100

This is a very common terminal and many third party terminals and PC terminal emulators offer a VT100 personality. A standard VT100 however has only 4 function keys and no soft font features. However it can do line drawing which can be used for [character boxes](TextTermCharBox.htm).

| KCML     | VT100 equivalent |
|----------|------------------|
| INSERT   | F1               |
| DELETE   | F2               |
| EDIT     | F3               |
| RECALL   | F4               |
| CONTINUE | CTRL-F           |
| EXECUTE  | CTRL-X           |
