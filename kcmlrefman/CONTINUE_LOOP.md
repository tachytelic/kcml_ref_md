CONTINUE LOOP <span style="font-size: 16pt ; ">command</span>

------------------------------------------------------------------------

General Form:\
\
     CONTINUE LOOP\
\

------------------------------------------------------------------------

The CONTINUE LOOP command will restart a program that has been stopped either with a [STOP](STOP.htm) or a [TRAP](TRAP.htm) statement or has been interrupted by a Ctrl+BREAK, and stop it again before the next [WEND](WHILE.htm) or [UNTIL](REPEAT.htm) statement which would complete the current [WHILE](WHILE.htm) or [REPEAT](REPEAT.htm) loop.

The CONTINUE LOOP command can be executed directly within the Workbench either as a single keystroke or from the toolbar.
