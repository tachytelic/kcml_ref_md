IBM3151 ASCII terminal

KTERM=ibm3151

This is the standard console screen on the IBM RS/6000. It has an AT style 12 function key keyboard but no soft font capability.

| KCML        | IBM3151 equivalent |
|-------------|--------------------|
| SF0         | Shift + CTRL + F12 |
| SF1         | F1                 |
| SF11        | Shift + F1         |
| SF21        | Shift + CTRL + F1  |
| EDIT/CANCEL | HOME               |
| HELP        | ESC + ESC          |
| INSERT      | Shift + Ins        |
| SHIFTINSERT | Shift+CTRL+Ins     |
| DELETE      | Delete             |
| SHIFTDELETE | Shift+CTRL+Del     |
| CONTINUE    | CTRL-F             |
| EXECUTE     | CTRL-X             |
