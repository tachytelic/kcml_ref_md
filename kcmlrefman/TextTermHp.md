HP 700/9x

KTERM=hp or KTERM=hpterm

On these terminals if a character is overwritten then the new character inherits the attributes of the previous character. This makes attributes unusable and so none are used by this implementation. Only 8 functions keys are available.

For these reasons it is recommended that the terminal is used in its EM100 or EM2200 personalities which emulate the vt100 and vt220 respectively.

Local printer support is not available in its native mode.
