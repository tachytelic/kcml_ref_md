\$HELPPARAMS

------------------------------------------------------------------------

General Forms :\
\
1.      \$HELPPARAMS = alpha_expression\
\
2.      alpha_receiver = \$HELPPARAMS\
\

------------------------------------------------------------------------

The \$HELPPARAMS statement is used to specify parameters. These are added to the URL that is used to display HTML help files specified by the \$HELP statement. The second form is used as a function returning the current contents of \$HELPPARAMS.
