TRUE

------------------------------------------------------------------------

General Form:\
\
     TRUE\
\

------------------------------------------------------------------------

The TRUE keyword is a numeric constant with a value of one. Any numeric variable can be set to TRUE:

numeric = TRUE

the variable can then be tested in a similar way:

IF (numeric == TRUE)\
     'get_record\
END IF

The TRUE keyword may also be used to create never ending loops which can be exited with a [BREAK](BREAK.htm) statement.

WHILE TRUE DO\
  .\
  .\
WEND
