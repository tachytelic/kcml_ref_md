\$TODAY

------------------------------------------------------------------------

General Form:\
\
     \$TODAY\

------------------------------------------------------------------------

The \$TODAY alpha function returns the current system date including the century. The date is returned with in the format "YYYYMMDD". The date is relative to the the users [timezone](timezone.htm) but is derived from the operating system date. The \$TODAY function is valid wherever an alpha expression is legal.

Syntax examples:

current\$ = TIME\
stamp\$ = DATE & TIME
