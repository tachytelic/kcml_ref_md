CONTINUE RETURN <span style="font-size: 16pt ; ">command</span>

------------------------------------------------------------------------

General Form:\
\
     CONTINUE RETURN\
\

------------------------------------------------------------------------

The CONTINUE RETURN command will restart a program that has been stopped either with a [STOP](STOP.htm) or a [TRAP](TRAP.htm) statement or has been interrupted by a Ctrl+BREAK, and stop it again immediately after the next [RETURN](RETURN.htm) statement that is associated with the most recent [GOSUB](GOSUB.htm) statement.

The CONTINUE RETURN command can be executed directly within the Workbench either as a single keystroke or from the toolbar.
