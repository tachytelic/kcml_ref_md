\$HELPINDEX

------------------------------------------------------------------------

General Forms :\
\
1.      \$HELPINDEX = alpha_expression\
\
2.      alpha_receiver = \$HELPINDEX\
\

------------------------------------------------------------------------

The \$HELPINDEX statement is used to specify either a directory name containing HTML files or the name of a tag within the HTML file specified by the \$HELP statement. The second form is used as a function returning the current contents of \$HELPINDEX.
