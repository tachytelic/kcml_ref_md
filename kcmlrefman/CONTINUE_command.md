CONTINUE <span style="font-size: 16pt ; ">command</span>

------------------------------------------------------------------------

General Form:\
\
     CONTINUE\
\

------------------------------------------------------------------------

The CONTINUE command is used to resume execution of a program that has been stopped with either a [STOP](STOP.htm) or a [TRAP](TRAP.htm) statement or has been interrupted by a Ctrl+BREAK. It is not possible to resume execution of a program if it is unresolved.

The CONTINUE command can be executed directly within the Workbench either as a single keystroke or from the toolbar.

See also:

[CONTINUE LOAD](CONTINUE_LOAD.htm), [CONTINUE LOOP](CONTINUE_LOOP.htm), [CONTINUE NEXT](CONTINUE_NEXT.htm), [CONTINUE RETURN](CONTINUE_RETURN.htm)
