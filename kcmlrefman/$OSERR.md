\$OSERR <span style="font-size: 16pt ; ">function</span>

------------------------------------------------------------------------

General Form:\
\
     \$OSERR\
\

------------------------------------------------------------------------

The \$OSERR function is used to return the native operating system error text of the most recent error. \$OSERR is valid wherever an alpha function is valid.

Error text is only generated by commands that invoke operating system calls, i.e. [LOAD](LOAD.htm), [SAVE](SAVE.htm), [OPEN \#](OPENhash.htm), [\$OPEN]($OPEN.htm) etc.

For example, if an attempt to [OPEN \#](OPENhash.htm) a native file that did not exist, the value of \$OSERR would be set to something like \`No such file or directory', [\$ERR](ERR$(.htm) would be set to \`File not found', and [ERR](ERR.htm) would be set to 82.

Unlike [ERR](ERR.htm) and [\$ERR](ERR$(.htm), \$OSERR retains the error text until another error is encountered. Errors which are not a result of a failed system call will leave [\$OSERR]($OSERR.htm) blank.

Syntax examples:

os_error\$(count) = \$OSERR\
FLD(error_description\$.err_text\$) = \$OSERR
