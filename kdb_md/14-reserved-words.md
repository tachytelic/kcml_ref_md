## Quoting Reserved Words as Identifiers

To use a reserved word as a column or table name, enclose it in double-quotes:

```sql
CREATE TABLE perms (
    "KEY"      VARCHAR(12),
    "USER"     VARCHAR(8),
    "MODULE"   VARCHAR(2),
    "PRIORITY" VARCHAR(1)
)

SELECT "DATE", "USER" FROM audit_log
```
