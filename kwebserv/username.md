Change User

You can change the your user ID using this link. You will be prompted for a new username and password with will be [authenticated](auth.htm) by the WebServer. The WebServer will then display the [Machine Type and Overview](overview.htm) page. Note that the [Current service](currentsrv.htm) may also change as a result.
