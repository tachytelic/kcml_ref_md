Connection Manager, kwebserv

The Connection Manager provides a platform independent method of managing Kerridge software. It provides a common mechanism for connecting different client products.\
This includes:

- Web Browsers
- KClient
- Kerridge ODBC Driver
- A SOAP client

The [Connection Manager](connmgr.htm) can handle multiple applications, known as [services](systemconf.htm#services), and provide security using access control lists based on user names and/or network addresses of incoming connections.

In addition to the Connection Manager, kwebserv also provides a common configuration interface in the form of the [Remote Web-based Administration tool](adminfns.htm). This allows the use of a Web browser to display and maintain system configuration information. There are also pages which provide useful diagnostic tools.
