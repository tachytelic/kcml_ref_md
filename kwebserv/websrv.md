Web Services

\
This is a look-up table which matches a connecting client program to a server.\
Each web service has a **name**. These names are used internally by the connection manager and should not be changed.\
When the connection manager invokes a server it will execute it under the specified **User** name.\
Next is the name of the **Server Program**. This should be an absolute pathname, such as */usr/lib/kcml/kcml* or *C:\Kerridge\KCML\kiodbc.exe*\
Finally there is a list of **Command Line Arguments**.
