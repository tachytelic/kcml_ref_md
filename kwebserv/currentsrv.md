Add/Change Current Service

Each [Valid user](systemconf.htm#validusr) can have a *Default service*, the value is stored in the *kconf.xml* configuration file. The default service's environment variables will be used for settings such as the localtion of [TERMFILE](mk:@MSITStore:kcmlrefman.chm::/EnvVars.htm#TERMFILE), the value of [SYSTEMID](mk:@MSITStore:kcmlrefman.chm::/EnvVars.htm#SYSTEMID) and for [directory listings](dirlist.htm), such as the **Application directory** and the [Panic file](mk:@MSITStore:kcmlrefman.chm::/EnvVars.htm#PANICDIR) directory. Any [Valid user](systemconf.htm#validusr) can change their default service using this link, provided that **WEBADMIN** has been set to **true**.
