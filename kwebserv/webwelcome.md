The Welcome page

This is the first page that the WebServer will display. There are two sets of links.

- **Kwebserv** [administration functions](adminfns.htm)
- **Connect to** [service ...](connserv.htm)

The **Kwebserv** [administration functions](adminfns.htm) link will allow you to view or update settings. This will require [authentication](auth.htm). After a username and password has been verified the WebServer will display the [Machine Type and Overview](overview.htm) page.\
The list of **Connect to** [service ...](connserv.htm) links will invoke KClient to connect to the specified service.

Finally, there are some links to Kerridge web-sites, [www.kerridge.com](http://www.kerridge.com) and [www.kcml.com](http://www.kcml.com). This is actually a HTML fragment, *kwroot/kwebserv/links.html*, which can be edited to include any other useful links.

View an example [welcome page](examples/welcome.htm).
