Directory Listings

Each directory listing will be shown in a table. The filename, size, modification date & time are shown. Users who are listed in the [Admin users](systemconf.htm#admusr) will also have a **Properties** column. This column has links to enable text files to be edited, and will show version numbers for KCML programs, KISAM files and Windows .exe and .dll files. The panic file directory listing will show the reason for the error occurring.

The filename is a link which will enable to file to be displayed. If the filename is not a link, then you probably don't have access to read the file.

The following types of file can be displayed.

- Text files
- HTML pages
- XML documents
- KCML Panic files, text and XML formats.
- KCML Screen dump files.
- GIF, JPEG & PNG images

If the WebServer cannot display the file then it will attempt to download it.

The column headings in a directory listing are links which allow the table to be sorted.
