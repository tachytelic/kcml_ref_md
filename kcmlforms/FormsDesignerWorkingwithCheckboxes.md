Forms Designer - Working with Check boxes

------------------------------------------------------------------------

To add a check box to the current form click on the <img src="bitmaps/form0035.gif" data-align="BOTTOM" data-border="0" alt="form0035.gif" /> icon from the controls palette.

Check boxes are very simple controls that allow an option to be set as either on or off. The user can check the box by simply clicking on the box section. The Check can later be removed again by re-clicking on the box.

Check boxes can also have an optional third state whereby the check box can be checked and grayed, this is enabled by setting the [*ThreeState* property on the](PROP_BUTTON_3STATE.htm) [Properties Tab](TheFormsDesignerPropertieslist.htm).

The caption text for a Check box is set with the

*Text\$* property. To include an accelerator key in the caption text simply insert an Ampersand before the required character.
