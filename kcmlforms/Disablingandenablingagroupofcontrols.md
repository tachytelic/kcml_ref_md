Disabling and enabling a group of controls

------------------------------------------------------------------------

To enable and disable the controls in a group the

*Enabled()* method is used. Passing a value of *TRUE* enables the controls in the group while a value of *FALSE* disables the control in the group. For example, the following would disable all controls in the group *grpControl1*:

.grpControl1.Enabled(FALSE)

To re-enable the controls in the group the following would be used:

.grpControl1.Enabled(TRUE)

Controls are grouped into named groups using the Forms Designer [Group tool](FormsDesignerGroupingControls.htm).
