The Grouping Menu - Make Group ... (Called from the Edit menu)\
Tool bar - <img src="bitmaps/the_0163.gif" data-align="BOTTOM" data-border="0" alt="bitmaps/the_0163.gif" />\
Shortcut - Ctrl+G

This option is used to group a number of controls together. This is particularly useful as it allows groups of controls to be operated on by the program. For example you may wish to disable an entire group, or make an entire group *ReadOnly*.
