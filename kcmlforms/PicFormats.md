Supported picture formats

The KCML client can render the following picture formats:

| Usual extension | Format | Comments |
|----|----|----|
| .PNG | PNG picture | Pronounced 'ping', this is a public domain replacement for the proprietary GIF format which compresses very well with no loss of definition and is suitable for most pictures. Introduced with KCML 6.00. |
| .JPG | JPEG picture | This produces the best compression for photographs but at a cost in definition. You can select how agressive the compression should be when saving the image. Introduced in KCML 5.02. |
| .BMP | Bitmaps | Both uncompressed and RLE encoded formats are supported. This gives the same definition as PNG but does not compress and is often very big so use only if no alternative. |
| .WMF | Windows metafile | Good for simple diagrams |
| .ICO | Windows icon | Limited sizes but usable on buttons and toolbars |

See also the [list of controls that support pictures.](PicEnabledControls.htm)
