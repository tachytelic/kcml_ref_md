Forms Designer - Changing the fonts used by a control

To change the font used by a control the

*Font* property for the control is modified. Once selected the font selection dialog is displayed (See below). When changing the font of a control you can either select one of the available stock fonts or you can create a new font object by entering the object name and setting the font properties. Stock fonts are preffered as they are based on the form font, therefore if the form font name is changed, or if a form font property is changed then the stock fonts will automatically be adjusted.
