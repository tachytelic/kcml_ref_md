Creating a form from the object browser

To add a new form object to the program currently in memory, simply call up the object browser (Ctrl+F), and select the [Forms Tab](TheObjectBrowserFormsTab.htm). From here click on [**New form object** and enter the name of the new form, or select an existing form from the list box. Next click on the](ObjectBrowserNewformobjectbutton.htm) [**Edit Form** button and the](ObjectBrowserFormEditbutton.htm) [KCML Forms designer](TheKCMLFormsDesigner.htm) will be displayed. Once you have completed designing the form close down the forms designer and return to the object browser.

Once a new form has been created, or an existing form selected, the [Forms Tab](TheObjectBrowserFormsTab.htm) within the object browser is then updated to contain details about the selected form.
