Update Tables Button

This button is used to update the list of available database tables. A new table is made available to the Forms Designer via the [Database Tab](TheObjectBrowserDatabaseTab.htm) in the KCML [Object browser](TheKCMLObjectBrowser.htm). Fields from newly added databases cannot be used in the Forms Designer until the list is updated. Once updated, the fields in a table can be viewed by double clicking on the table name. Individual fields can then be dragged onto the form.
