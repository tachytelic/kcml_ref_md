Forms Designer - Working with RTF controls

------------------------------------------------------------------------

To add an RTFcontrol to the current form click on the <img src="bitmaps/form0187.gif" data-align="BOTTOM" data-border="0" alt="form0187.gif" /> icon on the controls palette.

RTF controls allow the user to enter formatted text such as letters etc., into the control area. The program can then retrieve the text in RTF format and save it back to a file. Various programming methods and event handlers exist to make it easy to print the contents of the control and perform more complex operations such mail merges etc.

When a new RTF control is added to the form the default user toolbar is automatically available providing standard text formatting functions (***Bold****,* ***Italic***, ***Underline***, ***Bullets***, ***Indents*** and ***text justification***). To add the font size, font style and color selection boxes to the toolbar you must set the [*ToolFont* and](tmp/PROP_RICHEDIT_TOOLFONT.htm)

*ToolColor* properties to ***TRUE***.
