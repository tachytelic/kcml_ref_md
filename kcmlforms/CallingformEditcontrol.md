Calling another form instead of the drop down in a Edit control

The Edit control is normally used as a regular edit control with combo box style drop down capabilities. However, if the [*DropStyle* property is set to *Ellipsis* then the control displays ellipsis instead of a down arrow when the drop down portion is enabled. The Ellipses button if pressed calls the](tmp/PROP_EDIT_DROPSTYLE.htm)

*Click()* event handler which can then be used to display another form instead of the normal Edit drop down.
