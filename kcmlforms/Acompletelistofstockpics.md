Stock Pictures

Stock pictures can be used with the [Picture](tmp/PROP_GENERIC_PICTURE.htm) property aoociated with certain controls, for example the Picture control and grid control cells. The form an alternative to the programmable picture object. These stock pictures can be assigned both at design time within the Forms Designer or under program control. For example, the following would set a picture for the specified grid cell and picture control:

.gridControl1.cell(row, col).Picture = &.Information .picControl1.Picture = &.Satellite

To view a complete list available from KCML 6.00 you can run an example program which may be found in the page documenting the [Picture](tmp/PROP_GENERIC_PICTURE.htm#example) property.
