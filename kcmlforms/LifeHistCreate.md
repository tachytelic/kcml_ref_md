The Create() event

The [Create()](tmp/PROP_FORM_CREATE.htm) pseudo-event takes place entirely on the server before any data structures have been instantiated for the form. It allows the application to create controls for the form using the [.Import()](tmp/PROP_FORM_IMPORT.htm) or [.EditForm()](tmp/PROP_FORM_EDIT.htm) methods. See section on [dynamic forms](DynamicForms.htm).
