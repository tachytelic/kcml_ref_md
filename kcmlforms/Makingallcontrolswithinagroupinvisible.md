Making all controls within a group invisible

------------------------------------------------------------------------

To hide all the controls in a group the

*Visible()* method is used. Passing a value of *TRUE* means that the controls in the group are made visible while a value of *FALSE* hides the controls in the group. For example, the following would hide all controls in the group *grpControl1*:

.grpControl1.Visible(FALSE)

To make the controls visible again the following would be used:

.grpControl1.Enabled(TRUE)
