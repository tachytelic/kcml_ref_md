The Edit Menu - Undo option\
Shortcut - Ctrl+Z\
Tool bar - <img src="bitmaps/the_0284.gif" data-align="BOTTOM" data-border="0" alt="bitmaps/the_0284.gif" />

If selected this option will undo the previous action, for example if you accidentally delete some controls from the form, undo can be used to put them back. Alternatively, the File Menu - Cancel option can be used to return the form back to its original state prior to any changes that have been made.
