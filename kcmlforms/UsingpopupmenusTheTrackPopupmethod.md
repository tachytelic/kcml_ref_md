Using popup menus (The TrackPopup() method)

------------------------------------------------------------------------

Popup menus can be used with the KCML Grid control and KCML Picture button controls. Popup menus are created in the [menu editor](FormsDesignerMenubars.htm) within the [KCML Forms designer](TheKCMLFormsDesigner.htm). To display a popup menu the [TrackPopup()](tmp/PROP_MENU_TRACKPOPUP.htm) method is used. This method requires row and column co-ordinates to specify where on the form the menu is to be displayed. Both the KCML Grid and KCML Picture button controls return the row and column co-ordinates of the last mouse click in the [MouseX](tmp/PROP_BUTTON_MOUSEX.htm) and [MouseY](tmp/PROP_BUTTON_MOUSEY.htm) properties when the [Click()](tmp/PROP_PICBUTTON_CLICK.htm) event handler is triggered ([LeftClick(),](tmp/PROP_GRID_LEFTCLICK.htm) and [RightClick()](tmp/PROP_GRID_RIGHTCLICK.htm) for the grid control). These properties can then be used by the event handler to display a popup menu. Note that the [MouseX](tmp/PROP_BUTTON_MOUSEX.htm) and [MouseY](tmp/PROP_BUTTON_MOUSEY.htm) properties return the co-ordinates relative to the top left hand corner of the control, therefore the controls [Left](PROPNUM_X.htm) property should be added to MouseX and the [Top](tmp/PROPNUM_Y.htm) property should be added to MouseY. For example, the following picture button [Click()](tmp/PROP_PICBUTTON_CLICK.htm) event handler displays a popup menu onto the picture button surface:

\- DEFEVENT Form1.picControl1.Click() .menu1.TrackPopup(..Mousex + ..Left, ..MouseY + ..Top)) END EVENT
