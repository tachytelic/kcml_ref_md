FTP of pictures in the Forms Designer

This dialog is invoked by the Add Picture feature of the Tools menu. It allows files to be copied using the FTP protocol over a network connection from another machine to the workstation running the forms designer. They are then handled as local files using the name supplied. Files on the server can be browsed.

The server to use is determined by the FTP account selected. New accounts can be created by clicking the New Account icon.

<img src="bitmaps/FtpOpen.png" data-border="0" width="398" height="337" alt="FTP open dialog" />

Account

A drop down list of available FTP accounts. The account dtermines the server, the userid and password to use and the default directory on the server. A new account can be created with the New toolbar button. The directory on the server corresponding to the account is shown immediately below the combo. Below that is the list of files that match the type filter.
