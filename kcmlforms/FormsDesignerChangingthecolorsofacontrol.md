Forms Designer - Changing the colors of a control

------------------------------------------------------------------------

The colors of a control are changed by modifying the [*BackColor* and](tmp/PROP_GENERIC_BACKCOLOUR.htm)

*TextColor* properties. Once selected the Color change dialog is displayed (See below). To change the color of a control simply select one of the stock color objects from the list. Note that new color objects can be created by entering the object name at the Color Name prompt. A specific color can then be assigned to the new color object by clicking on the required color from the basic colors or custom colors palette. Alternatively a system color can be selected from the system colors list.
