KCMLEdit control overview

The KCML Edit control is an enhanced version of the standard Windows edit control. It has extra functionality and uses less client resource. The extra functionality includes.

- Automatic database integration
- Controls can be typed as currency, dates etc.
- Controls can be labelled
- Support for dropdown lists and ellipsis events
- Sophisticated validate events
- Common functionality with editable grid cells
- Eurocurrency support
- Automatic alignment within edit groups

While the standard [edit control](IntroEdit.htm) is supported by KCML its use is strongly discouraged and you should use the KCMLEdit control for all text input requirements.

KCML edit controls generally benefit from being part of an [Edit Group](IntroEditGroup.htm).

For a complete list of the methods, properties and events supported see the [control summary](tmp/kcmledit.htm) in this section.
