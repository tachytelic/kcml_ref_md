Creating new Event Handlers

Event handlers are normally created by the [Forms designer](TheKCMLFormsDesigner.htm) or by the [Object Browser](TheKCMLObjectBrowser.htm), however it is also possible to directly enter them by typing in the [DEFEVENT ... END EVENT](DEFEVENT.htm) pair. To ensure that programs follow a structured format it is recommended that event handlers are created within the form definition from which they are attached.
