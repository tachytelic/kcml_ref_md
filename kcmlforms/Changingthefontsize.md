Changing the font size

The font size for a font set can be changed by setting the

*Size* property. For example:

.dlgfont1.size = 24

This property can also be used to return the current size of the font for a font set, for example:

Size = .dlgfont1.size

Note that you cannot change the size of a stock font as it is based on the size of the font used by the form.
