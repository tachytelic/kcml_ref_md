The Language Menu - Add Language Option\
Toolbar - <img src="bitmaps/the_lang.gif" data-align="BOTTOM" data-border="0" alt="bitmaps/the_lang.gif" />

This option is used to add an additional language to the form. You can then choose the required language from the language selection box and modify any text properties for the selected language. When the form is executing byte 20 of the \$OPTIONS RUN system variable can be used to select the required language.
