Tools Menu - Tab Stops\
Tool bar - <img src="bitmaps/tool0314.gif" data-align="BOTTOM" data-border="0" alt="tool0314.gif" />

Normally when a user presses the TAB key, focus is moved to each control in the order in which they were added to the form. This order can be changed with this option. Once selected the form is redisplayed and normal form editing features are disabled. Each control is displayed with the letter "T" next to the control clicking on the control selects and deselects the control from the TAB order.
