The Edit Menu - Set Tab order option\
Shortcut - Ctrl + D\
Tool bar - <img src="bitmaps/the_0155.gif" data-align="BOTTOM" data-border="0" alt="bitmaps/the_0155.gif" />

When a form is displayed the TAB key is used to move to the next control on the form. Normally the TAB key will move through the controls in the order in which they were placed on the form. This option is used to change this order. Once selected, the controls on the form are displayed along with their current TAB sequence number. To set a new tab order simply click on the controls in the order required.

This option is also available by right clicking on the form.
