Grid control overview

Grid controls are arrays of grid cells best suited for displaying and editing tabular data. The cells have much the same functionality as KCMLEdit masked edit controls. In particular grids are database enabled and can be populated automatically from a database result set.

Properties, events and methods can apply to the whole grid or at the cell level or both.
