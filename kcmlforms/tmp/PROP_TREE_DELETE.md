**Used to drop all the nodes of a tree**

This method deletes and frees all the nodes in the tree. It is applied directly to the control or to an object reference to the control. .treeControl1.Delete()

It is polymorphic with the [Delete()](PROP_TREEITEM_DELETE.htm) method which deletes tree nodes.
