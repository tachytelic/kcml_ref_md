**Called when the maximum number of characters set by limittext is reached**

This event is triggered when the user reaches the previously set maximum text length for the edit control. This maximum was set using the [LimitText](PROP_EDIT_LIMITTEXT.htm) property thus: .EditControl1.LimitText = 40
