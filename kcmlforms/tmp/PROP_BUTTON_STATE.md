**Determines or sets the state of a checkbox**

This property is used to determine or set the state of a Check box. Returns *TRUE* if the check box is filled. For example, the following could be used to determine the state of the check box: BoxState = .cbControl1.State

<span style="font-family: Courier New,monospace; "> </span>If the Check box is [*ThreeState* check box then the this property will return *TRUE* if the check box is checked and a value of 2 if the box is grayed.](PROP_BUTTON_3STATE.htm)
