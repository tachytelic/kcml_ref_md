**Specifies the number of lines per row**

This property is used to specify the number of text lines that are to appear in each row in a grid control. To add multiple lines of text to a grid cell the [*Type\$* property for the relevant cells should be set top allow multiline strings](PROPSTR_TYPE.htm)

 

##### Example:


     n = .gridControl.LinesPerRow
