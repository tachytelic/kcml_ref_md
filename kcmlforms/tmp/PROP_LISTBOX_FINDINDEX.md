**Used to return the index associated with the specified string**

***Str***      A valid string within the control

Used to return the index number associated with the specified string. If the list box entries were added with tags (see [*Add()*) then the index of the specified tag is returned. For example: Ind = .listControl1.GetIndex(item\$)](PROP_LISTBOX_ADD.htm)
