*(Left, Center, Right)*

**Allows the alignment of text to be set for multiline edit controls**

Used to specify the alignment of the text within an Edit control. The available settings are ***Left,*** ***Center*** and ***Right***. This is a style property and it cannot be set at run time after the Create() event.
