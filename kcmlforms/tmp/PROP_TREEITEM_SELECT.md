**Select this tree item**

This method is used in conjunction with a tree item object or the [Item()](PROP_TREE_ITEM.htm) method to select a node in a tree control. The selection is indictated by the node changing color. A [SelChange()](PROP_TREE_SELCHANGE.htm) event is triggered by the selection process. .treeControl1.Item(NodeIndex).Select()
