**Turns on/off the use of an Italic font**

If set to *TRUE*, the associated font set will use an italic font. For example, the following could be used to change the font specified by the *dlgfont1* font set to an italic font: .dlgfont1.italic = TRUE

and to return the current setting the following could be used: Type = .dlgfont1.italic
