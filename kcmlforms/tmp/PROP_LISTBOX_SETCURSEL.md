**Used to select items within a list box**

This method is used to position the cursor selection line to the specified index within the list box. For example: .listControl1.SetSelection(10)

This method can also be used without any arguments to clear any current selection

Note that the [*AddSelection()* method is used to select items in multiple and extended selection list boxes.](PROP_LISTBOX_ADDSELECTION.htm)
