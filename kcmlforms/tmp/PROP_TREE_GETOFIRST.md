**Get object for first item**

This property returns an object reference to the first tree node in the tree. The property can be used with the [Next](PROP_TREEITEM_GETONEXT.htm) and [Child](PROP_TREEITEM_GETOCHILD.htm) properties to recursively navigate the entire tree. OBJECT c = .treeControl1.First
