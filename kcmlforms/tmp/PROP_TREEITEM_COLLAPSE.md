**Force collapse of tree item**

This method is used in conjunction with the [*Item()* method to collapse levels within a tree control. The](PROP_TREE_ITEM.htm) [*ExpandChange()* event is called each time a level is expanded and again when a level is collapsed.](PROP_TREE_EXPANDCHANGE.htm)

Example: .treeControl1.Item(ItemToCollapse).Collapse()

Tree levels can be expanded again with the [*Expand()* method.](PROP_TREEITEM_EXPAND.htm)
