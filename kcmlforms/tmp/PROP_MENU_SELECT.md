**Called when a menu item is selected**

This event handler is called when the specified menu option is selected. This event handler is automatically created for menu options by the [Menu Editor](/FormsDesignerMenubars.htm) if the "Create KCML Event" option is selected.

##### Example:


     DEFEVENT Form1.menuitem.Select()
