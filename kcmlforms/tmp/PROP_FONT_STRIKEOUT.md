**Turns on/off strike through**

If set to *TRUE*, the associated font set will display the font with a horizontal line through it. For example, the following could be used to change the font specified by the *dlgfont1* font set to use the *Strikeout* style: .dlgfont1.Strikeout = TRUE

and to return the current setting the following could be used: Type = .dlgfont1.Strikeout
