**Set to indicate item is expanded**

This property is used within the [*ExpandChange()* event handler to determine if a tree branch is being expanded or collapsed. A value of ***TRUE*** means that the branch is being expanded. For example: BranchStatus = .treeControl1.Item(.treeControl1.ExpandItem).Expanded](PROP_TREE_EXPANDCHANGE.htm)
