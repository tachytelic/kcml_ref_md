**Specifies the number of rows**

This property is used to specify the number of rows in a Grid control, for example: .GridControl1.rows = 10

<span style="font-family: Courier New,monospace; "> </span>Increasing the number of rows will add additional blank rows to the end of the grid. Decreasing the number of rows will cause cells beyond the new number of rows to be deleted.
