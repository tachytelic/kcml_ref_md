**Instructs the control to echo an asterisk as each character is entered**

If set to *TRUE* then the edit control will echo an asterisk as each character is entered. This is a design time only parameter that has no effect if changed under program control. Only available for single line edit controls, i.e. if [MultiLine](PROP_EDIT_MULTILINE.htm)is set to *FALSE*. The password character can be changed with the [PasswordChar](PROP_EDIT_PASSWORDCHAR.htm) property.
