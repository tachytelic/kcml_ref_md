**Specifies the body color**

This defines the basic color of the arrow. The color of the edging is set with the [.EdgeColor](/tmp/PROP_ARROW_EDGECOLOUR.htm) property. For more about setting colors see the generic [.TextColor](PROP_GENERIC_TEXTCOLOUR.htm) property and the page on [stock colors](../Acompletelistofstockcolors.htm).

##### Example:


     .arrow.BodyColor
