**Used to test to see if an item in a multiple selection list box is selected**

***Int***      The index of the item to test

Used to test to see if an item in a multiple selection list box is currently selected. Returns *TRUE* if the item is selected. For example: IF (.listControl1.GetSelected(ItemNo)) ... END IF
