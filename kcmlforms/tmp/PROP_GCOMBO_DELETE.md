**Used to remove individual items from a DBEdit dropdown**

***Int***      Specifies the index of the item to be deleted

Used to remove individual items from an Edit control dropdown within a grid cell. For example to remove the fifth item from the drop down the following would be used: .gridControl1.Delete(5)
