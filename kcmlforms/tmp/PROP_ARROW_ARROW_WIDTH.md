**Width of arrow point**

This property specifies the width of the arrow head in dialog box units. The head can never be narrower than the the line of the arrow.

<img src="/bitmaps/arrowwidth.png" data-align="BOTTOM" data-border="0" width="250" height="200" alt="arrow width" />

##### Example:


     n = .arrow.ArrowWidth
