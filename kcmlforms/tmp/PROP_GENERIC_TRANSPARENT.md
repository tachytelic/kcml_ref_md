**Makes the controls background transparent**

If set to *TRUE* the control will set its background to a transparent state. Therefore the background color or picture currently being used by the form will be used as the background of the control. This property would normally be set to ensure that the default background of the control always matches the background of the form. For example: .cbControl1.Transparent = TRUE
