**Determines if the picture background is to be filled**

This property is used to determine if the picture button background is to be filled. If set to *FALSE* then the control area is filled in with the current background color, otherwise the button background is not displayed. For example: .picControl1.NoFill = TRUE

and to test for the current setting the following could be used: Fill = .picControl1.NoFill
