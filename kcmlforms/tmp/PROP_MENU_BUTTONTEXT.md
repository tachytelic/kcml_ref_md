**Specifies the label used by the button associated with a menu item**

This sets the text of a toolbar button independently of the text of the menu option. Ideally this text should be kept short. Buttons will expand to accomadate the text but only up to the point where all the buttons will remain visible. .FileMenu.New.ButtonText\$ = "New"
