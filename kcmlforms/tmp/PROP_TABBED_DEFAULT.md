**Specifies the default tab for the tab control**

This property is used to set the specified tab as the default tab within a tab control. The default tab is the tab initially displayed when the form is opened. A value of *TRUE* makes the specified tab the default tab. For example: .TabControl1.tab1.default = TRUE
