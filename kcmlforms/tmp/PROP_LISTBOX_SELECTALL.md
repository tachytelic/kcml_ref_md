**Used to select items in an extended or multiple selection list box**

In a multiple selection or extended selection list box, this method will select all the items in the list. For example: .listControl1.AddSelection()

This is a special case of the more general [AddSelection(int)](PROP_LISTBOX_ADDSELECTION.htm) which selects a specific item in the list by index.

To remove all current selections use the [SetSelection()](PROP_LISTBOX_NOCURSEL.htm) method without any parameters.
