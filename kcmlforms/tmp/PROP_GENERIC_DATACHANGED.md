**Determines if the data in the control has changed**

This property is set to *TRUE* by KCML if the information in the control has changed since the control was displayed. For example: Changed = .Edit1.DataChanged

The *DataChanged* property can also be used to set the *DataChanged* state of a control. For example: .Edit1.DataChanged = TRUE
