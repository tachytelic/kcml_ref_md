**Set if the menu will be used as a propup menu**

KCML supports two types of menus, the main menu at the top of the form and an arbitrary number of popup menus which are associated with the form and which can be invoked on demand with the [TrackPopup()](PROP_MENU_TRACKPOPUP.htm) method. This style bit property is set TRUE in the forms designer to mark a menu as a popup menu. It is FALSE for the main menu.

See [Working with menus in the Forms Designer](/FormsDesignerMenubars.htm) for more about how popup menus are created.
