**Specifies the default echo character for passworded DBedit controls**

Used to specify the echo character used to replace each character as it is entered into the KCML edit control. This property can be set both at design time and under program control. This is only available for single line edit controls. For example: .EditControl1.PasswordChar\$ = "#"
