**Create a new tab page on this tab control**

This method is used to create a new tab page and append it to the tab pages on the tab control. This method can only be used in the Create() event. See [CreateControl()](/tmp/PROP_FORM_CREATECONTROL.htm) for more details and examples.

##### Example:


     .tabControl.CreateTabPage(a$)
