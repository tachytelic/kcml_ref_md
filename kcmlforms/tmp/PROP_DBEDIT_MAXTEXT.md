**Called if the number of characters set by LimitText is reached**

This event is triggered when the user reaches the previously set maximum text length for the KCML edit control. This maximum was set using the [LimitText](PROP_DBEDIT_LIMITTEXT.htm) property thus: .DbEditControl1.LimitText = 40
