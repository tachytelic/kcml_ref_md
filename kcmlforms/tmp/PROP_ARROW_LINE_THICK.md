**Thickness of the main line part of the arrow**

This property specifies the thickness of the main line part of the arrow measured in dialog box units.

<img src="/bitmaps/arrowthick.png" data-align="BOTTOM" data-border="0" width="250" height="200" alt="Line thickness" />

##### Example:


     n = .arrow.LineThickness
