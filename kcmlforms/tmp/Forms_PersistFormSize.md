PersistFormSize method

<img src="/bitmaps/browsetool22.png" data-border="0" width="16" height="15" alt="method icon" /> PersistFormSize(Persist)

The **PersistFormSize()** method allows the user to set the behaviour of the sizing of resizable forms. The default behaviour is off, but it can be enabled and reset using the single boolean parameter to this method. If enabled, then new top-level resizable forms will maintain the same window size and position as the previous top-level window, including whether the form is maximized or not. (However, the form size will never be smaller than its minimum design time size). forms.PersistFormSize(TRUE)
