**Called when the user clicks on an item in the list**

This event is called if the user clicks on an item in a list box. The index number of the selected string can be determined with the [Index](PROP_GENERIC_DATAPTR.htm) property.

##### Example:


     DEFEVENT Form1.listControl.Click()
