**Automatically adds a vertical scroll bar when required**

This property can only be set for a multi line edit box, that is an edit control that has the [MultiLine](PROP_EDIT_MULTILINE.htm) property set. If *AutoVscroll* is set to ***TRUE***, it allows the user to scroll the text of the control vertically while always having the insertion point visible. This property is a design time only property that has no effect if changed under program control.
