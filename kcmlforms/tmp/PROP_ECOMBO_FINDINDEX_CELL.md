**Used to return the index number associated with the specified string**

***Str***     A valid string within the control

Used to return the index number associated with the specified string. If the list entries were added with tags (see [*Add()*) then the index of the specified tag is returned. For example: Ind = .EditControl1.GetIndex(item\$)](PROP_ECOMBO_ADD.htm)
