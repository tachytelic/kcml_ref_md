**Called when the user double clicks on an item in the list**

This event is called if the user double clicks on an item in the Combo ox. Note that the [*Click()* event will have been called prior to this event.](PROP_COMBO_SELCHANGE.htm)

 

##### Example:


     DEFEVENT Form1.comboControl.DblClk()
