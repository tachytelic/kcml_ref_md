**Set of item clicked**

In a [Click()](PROP_GRAPH_CLICK.htm) event handler, the ClickSet property contains the index of the data item clicked on (see [Data](PROP_GRAPH_DATA.htm)).

##### Example:


     n = .graphControl.ClickSet
