<img src="/bitmaps/browsetool22.png" data-border="0" width="16" height="15" alt="method icon" /> GetTag\$(int) Method

**Used to return the tag associated with the specified index value**

***Int***      The index of the required item

Used to return the tag from the specified index location. For example, the following would return the tag from the fifth item added to the combo box: Tag\$ = .comboControl1.GetTag\$(5)
