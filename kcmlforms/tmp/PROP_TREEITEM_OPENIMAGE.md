**Image shown when item is expanded**

This property is used to specify the image used by selected or expanded items within a tree control. If no image is specified then the default file and opened folder icons are used. The unselected, unexpanded image is specified with the [*Image* property. .treeControl1.Item(Count).Image = &.CD .treeControl1.Item(Count).OpenImage = &.CD](PROP_TREEITEM_IMAGE.htm)
