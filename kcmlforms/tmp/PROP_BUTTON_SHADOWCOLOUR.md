**Specifies the buttons shadow color**

This property is used to specify the color that is to be used by the shadow section of a picture button. This property is on relevant if the [*Thickness* property is set to a value other than zero. To change the shadow color within a program an alternative color set is specified, for example: .picControl1.ShadowColor = &.DarkBlue](PROP_BUTTON_THICKNESS.htm)
