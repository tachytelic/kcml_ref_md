**Enables/disables readonly mode for the control**

This Boolean property can be used to set the listbox into a read only mode in which the user cannot select items with the mouse or keyboard. .listControl1.ReadOnly=TRUE

The following could be used to determine the current *ReadOnly* state of the control: State = .listControl1.ReadOnly
