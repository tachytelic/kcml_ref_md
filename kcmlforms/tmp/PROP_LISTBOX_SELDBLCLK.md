**Called when the user double clicks on an item in the list**

This event is called if the user double clicks on an item in the list box. Note that the [*Click()* event will have been called prior to this event.](PROP_LISTBOX_SELCHANGE.htm)

 

##### Example:


     DEFEVENT Form1.listControl.DblClk()
