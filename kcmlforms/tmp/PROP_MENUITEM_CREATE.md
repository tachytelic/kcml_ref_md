**Create a menu item as a child of this menu item**

This method is used to add sub menu items to a menu item. This method can only be used in the Create() event. See [CreateControl()](/tmp/PROP_FORM_CREATECONTROL.htm) for more details and examples.

##### Example:


     .menuitem.CreateMenuItem(a$)
