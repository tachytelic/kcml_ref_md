**Specifies the default background color**

This property allows an alternative background color to be specified for a control. The new color is specified by assigning a pointer to one of the available [stock colors](/Acompletelistofstockcolors.htm), alternatively a color set created by the forms designer can be specified. For example: .EditControl1.BackColor = &.DarkBlue
