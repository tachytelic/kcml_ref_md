**Form has caption**

If set to *TRUE* the form’s title bar is made visible. Setting this property to *FALSE* will remove the title bar however you also need set the [*Text\$* property for the form to blank. Removing the title bar will also remove the forms system menu and the minimize and maximize options if set.](PROPSTR_TITLE.htm)
