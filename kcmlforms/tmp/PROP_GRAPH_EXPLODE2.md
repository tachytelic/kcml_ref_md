**Show graph item in exploded form**

This is a simplified form of [Explode()](PROP_GRAPH_EXPLODE.htm) which can be used if there is only one data set. It is equivalent to Explode(0, index, explode).

##### Example:


     .graphControl.Explode(i, j)
