Selecting a table from the object browser

To add a new database table object to the program currently in memory, simply call up the object browser (Ctrl+F), and select the [Database Tab](TheObjectBrowserDatabaseTab.htm). From here click on

**New Table**, this will then display a list of tables available for the specified database connection type (the default is KISAM). If no database tables exist in this list then there is a configuration problem, see the Database section later on in this document for more information. From the list of available tables select a table and details of the tables columns will be displayed on the right hand side. Selecting **OK** will add the table definitions to your program. Notice that a definition for the connection type is also created. It is recommended that you select all of the tables that are likely to be used by the form before you start to design the form itself.

The database tab within the object browser also allows you to obtain more information about the selected database table. By selecting a radio button from the list the additional information can be displayed.
