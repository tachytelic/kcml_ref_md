<span id="International_forms" international_forms=""></span>International Forms

The KCML Forms Designer allows multiple strings to be entered for each text field on a form. This allows strings of different languages to be stored within the form. The required language is then later selected by the program. Multi-lingual strings can also be used directly within the program by the use of what are called Chevrons. See [Programming multi-lingual strings](Programmingmultilingualstrings.htm) for more information.
