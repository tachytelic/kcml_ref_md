Generic methods

Form objects and controls have a number of generic methods used with the item that are available directly from within KCML programs. Some methods have not been made directly available as they are considered to be irrelevant to KCML. However, there is a generic method available for all form controls to allow all Windows messages to be sent to the form. This method is called [*Message()* which basically follows the format of the Windows](tmp/PROP_GENERIC_SENDMI.htm)

*SendDlgItemMessage()* call.
