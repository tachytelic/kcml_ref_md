Invoking the Forms Designer

In the workbench you can invoke the Forms Designer for the first time either by right clicking in the editor window for a blank program or by using the [Forms Browser](mk:@MSITStore:workbench.chm::/wbbrowser.htm) menu which has options to Create a form or to edit the current form selected in the browser. You can also edit an existing form by right clicking on its DEFFORM definition and chosing Edit Form from the context menu.

In the original 5.02 editor, you can call the Forms designer via the [**Edit** button on the](ObjectBrowserFormEditbutton.htm) [Object Browser’s](TheKCMLObjectBrowser.htm) [Forms tab](TheObjectBrowserFormsTab.htm). Once you have designed a form and returned to the Editor window you can recall the Forms Designer by right double clicking on any forms object definition DEFFORM in the program.
