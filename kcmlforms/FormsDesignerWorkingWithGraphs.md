Forms Designer - Working with Graph controls

------------------------------------------------------------------------

To add an Graph control to a form click on the <img src="bitmaps/GraphTool.png" data-align="BOTTOM" data-border="0" alt="Graph control tool" /> icon on the controls palette, then click on the form to outline the rectangle that will contain the graph.

The controls supports various 2D and 3D pie and bar graph presentations through the [.Type](tmp/PROP_GRAPH_TYPE.htm) enumeration. This can be set at design time.

For more about the controls capabilities and using graphs at runtime see the [Introduction to Graph Control](IntroGraph.htm) section in the controls reference. For a complete list of the methods, properties and events supported see the [control summary](tmp/graph.htm) in that section.

The KCML Graph Control was added for KCML 5.03 and enhanced for KCML 6.00. For compatibility issues with the GraphServer OCX see the [compatibility notes](GraphCompat.htm).
