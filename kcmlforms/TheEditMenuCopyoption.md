The Edit Menu - Copy option\
Shortcut - Ctrl + Insert\
Tool bar - <img src="bitmaps/the_0286.gif" data-align="BOTTOM" data-border="0" alt="bitmaps/the_0286.gif" />

This option is used to copy the currently selected controls into the Windows clipboard. The [Paste](TheEditMenuPasteOption.htm) option can be used to paste the controls elsewhere onto the form.
