Supported Controls

The Forms designer has a number of standard built in controls (See [KCML Standard controls](KCMLFormcontrols.htm)), these are displayed on the tear off controls palette at the base of the screen.

<img src="bitmaps/controlpalette.png" data-border="0" alt="Control palette shoing OCX controls on the right" />

Additional OCX controls can be installed via the [Install OCX](TheFileMenuInstallOCXoption.htm) option from the File Menu.
