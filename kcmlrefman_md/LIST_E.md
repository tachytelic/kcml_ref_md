# LIST E

> Lists all local KCML environment variables set by `ENV(`.

## Syntax

```
LIST [title] E
```

## Description

Displays the name and value of every environment variable that has been set (or modified) in the current KCML session using the `ENV(` function. This is useful for debugging which env vars have been set programmatically.

## See Also

- `ENV(` — read or set an environment variable
- `EnvVars` — reference for all KCML-recognised environment variables
