# WDW / Windows DW Terminal

> KCML terminal configuration for the Kerridge Windows DW terminal emulator.

## Description

`KTERM=wdw`

The Windows DW emulator provides both VT220 emulation and native extended KCML text mode. Supports:

- Full box and block graphics
- 132-column mode
- Attributes (bold, blink, reverse, underline)
- Color
- Local printing

Used when running KCML from Windows without KClient in text mode.
