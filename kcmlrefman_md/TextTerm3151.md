# IBM 3151 ASCII Terminal

> KCML terminal configuration for the IBM 3151 terminal.

## Description

`KTERM=ibm3151`

Standard console on the IBM RS/6000 (AIX). AT-style keyboard with 12 function keys; no soft font capability.

### Keyboard mapping

| KCML key | IBM 3151 key |
|----------|-------------|
| SF0 | Shift+Ctrl+F12 |
| SF1 | F1 |
| SF11 | Shift+F1 |
| SF21 | Shift+Ctrl+F1 |
| EDIT/CANCEL | HOME |
| HELP | ESC+ESC |
| INSERT | Shift+Ins |
| SHIFT INSERT | Shift+Ctrl+Ins |
| DELETE | Delete |
| SHIFT DELETE | Shift+Ctrl+Del |
