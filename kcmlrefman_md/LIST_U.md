# LIST U

> Lists all available user-written C CALL routines with their argument signatures.

## Syntax

```
LIST [title] U
```

## Example output

```
:LIST U
R7_DATE2J
     str TO int
R7_J2DATE
     int, int TO str
```

## See Also

- `CALL` — call a user-written C routine
- `LIST CALL` — find references to CALL in source
