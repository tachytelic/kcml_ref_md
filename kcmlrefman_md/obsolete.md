# Obsolete Statements

> KCML statements that have been removed or are no longer meaningful.

## Dropped statements

| Statement | Notes |
|-----------|-------|
| `$REAL` | DOS KCML up to version 4.0: forced variables into the first 640 KB of real memory for exchange with external C routines. Has no effect on Unix or Windows versions of KCML. |
