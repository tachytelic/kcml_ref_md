# <span id="savingabookmark"></span> Saving a bookmark

The user creates a bookmark icon by choosing the bookmark option from the [system menu](systemmenu.htm) on the current form. This brings up a dialog box with a suggested name for the new connection (taken from the current [\$BOOKMARK](mk:@MSITStore:kcmlrefman.chm::/$BOOKMARK.htm) value) and an icon that can be dragged with the mouse and dropped onto the desktop. Alternatively you can click on the button to save an icon on the Windows Desktop. Once on the desktop it can be moved to a folder. The suggested name comes from the current value of \$BOOKMARK and may not be blank.
