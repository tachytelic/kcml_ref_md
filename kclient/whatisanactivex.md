# <span id="whatisanactivex."></span> What is an ActiveX?

An ActiveX is an Interactive COM object that can be embedded in a container program such as a Web browser and can interact with a user. ActiveX controls are supported only on Windows 95/98 and NT by Internet Explorer version 3 and later. Netscape can support them via third party plug-ins such as [ScriptActive](http://www.ncompasslabs.com).

ActiveX objects can be automatically downloaded over the Internet. A secure digital signing process guarantees that the object is safe and has not been tampered with.
