# <span id="systemmenu"></span> System menu

The client can be customized when running via the **system menu**. This is invoked by left clicking on the top left icon on the title bar or by right clicking the title bar of a form or the text window. The top section of the menu contains standard Windows functions for minimizing, restoring, moving and resizing.

Below the separator line are Kclient specific options for forms

| Menu option | Purpose |
|----|----|
| [Preferences](preferences.htm) | A tabbed dialog used to set Forms and Text preferences as well as control the cache settings |
| [Bookmarking…](whatisabookmark.htm) | A drag and drop method for saving positions within applications. It requires that the application specifically support this feature. |
| About… | A standard about box. |

\
