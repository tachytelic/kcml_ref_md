# <span id="connectionpage"></span> Connection page

Depending on what is chosen from the first drop down list box, this page will be displayed in two layouts

| Connection type | Purpose |
|----|----|
| [Connect locally](localconnection.htm) | This will only be available for WKCML where both the client and server are on the same machine. |
| [Connect to remote server.](remoteconnection.htm) | Available for client server versions of KCML where the client connects over a TCP/IP network to the Unix or NT server. |
| [Connect with encryption.](remoteconnection.htm) | KClient connects to a remote server as above but uses the Transport Layer Security (TLS/SSL) protocol to provide a secure encrypted connection. See [Secure Network Connections](mk:@MSITStore:kwebserv.chm::/secureconn.htm). |
