# <span id="fonts"></span> Fonts

The font used for text can be changed. By default Kclient will use the FixedSys font in whatever size is supplied. It is possible to choose another monospaced font using the Change Font button and dialog. This is a standard Windows Common Dialog and offers a choice of font styles. It is currently not possible however to choose other than a regular font and if you choose bold or italic the choice will be ignored. Note that only monospaced fonts are displayed as proportionally spaced fonts would not line up in text mode.

Do not choose the *kclient* font which will be automatically invoked in text mode when in the alternate character set. This is a symbol font that only contains a limited number of characters for graphics.
