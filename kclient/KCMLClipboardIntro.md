Programmed access to the clipboard

Kclient (and WDW before it) has intrinsic \$DECLARE functions that allow programs to manipulate data in the clients clipboard.

|  |  |
|----|----|
| [KCMLWriteClipboard](KCMLWriteClipboard.htm) | Write information into the Windows clipboard. |
| [KCMLReadClipboard](KCMLReadClipboard.htm) | Read information from the clipboard. |
| [KCMLGetClipboardLength](KCMLGetClipboardLength.htm) | Retrieve the size of the clipboard |
