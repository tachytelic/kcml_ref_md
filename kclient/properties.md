# <span id="properties"></span> Properties

| Property | in/out | Purpose |
|----|----|----|
| Source | \[in\] BSTR | Allows the text of a program to be specified in the calling script. Only valid for a local connection. |
| Connect | \[in\] BSTR | Command line arguments. |
| Download | \[in\] BOOL | If set then the kclient will not be started but instead a Downloaded() event will be triggered. |
| OwnBackground | \[in\] BOOL | Set if container will paint the background for forms |
| Title | \[out\] BSTR | Returns the current window title |
