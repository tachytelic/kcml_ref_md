KI_CLEAR_HANDLES

This can be used at any time to close and free all the temporary autoallocated handles. See [KI_ALLOC_HANDLE](KI_ALLOC_HANDLE.htm) for how to autoallocate a temporary handle. Normally explicit calling of this function should be unnecessary as temporary handles will be implicitly closed and freed on [RUN](mk:@MSITStore:kcmlrefman.chm::/RUN.htm) or [LOAD](mk:@MSITStore:kcmlrefman.chm::/LOAD.htm).
