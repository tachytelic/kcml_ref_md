## Line Menu

These options are used with line numbers

<img src="bitmaps/linemenu.png" data-align="BASELINE" data-border="0" width="206" height="91" alt="line menu" />

### Line numbers

Clicking this will toggle on or off the display of line numbers in the editor and debugger. By default they are not displayed.

### Insert line number

Inserts a new line number before the current statement. The new line number is generated to fall within the range of the previous and next line numbers.

### Delete line number

Deletes the line number preceding the current statement. Note that line number references are not automatically modified by this action.

### Edit line number

Allows the line number preceding the current statement to be modified. The new number must fall within the range of the previous and next line numbers. Note that line number references are not automatically modified by this action.

### Delete all line numbers from block

Deletes all line numbers within the currently marked block of program text. Note that line number references are not automatically modified by this action.
