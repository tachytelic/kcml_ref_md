## New features

New features include the browser on the right. The first three toolbar buttons allow you to browse forms, objects, options (largely unfunctional) and files. There is a new event log window and it is possible to set traps on controls and properties (making it very easy to track where things are being done to a control). The scroll bar is functional and if you have a mouse wheel (once you have one you will never know how you managed without) then that will work as well. The suggest option (right click in the editor) is also available as ALT-CursorDown (to see what is does, try typing gb\_ and then hit ALT-CursorDown - a list of all variables beginning gb\_ will appear). If you break in with a form up so that it appears dark grey, then clicking on controls will automatically hightlight them in the forms browser.
