## Window Menu

These menu options allow you to switch windows and to display other dialogs.

<img src="bitmaps/windowmenu.png" data-align="BASELINE" data-border="0" usemap="#windowmenu" width="216" height="381" alt="Window menu" />

### Show edit window

Obsolete, to be removed.

### <span id="global window">Show global window</span>

Toggles the currently selected global partition into the editor window. The window title will change to say GLOBAL. Also available by pressing the F8 function key shortcut.

### <span id="function headers">Show function headers</span>

Changes the display to show subroutine headers only. The body of the routines is suppressed allowing an easy overview of the structure of the program. See [Function Headers](Function_Headers.htm) section.

### <span id="execute window">Show execute window</span>

Switches to the current form or, in text mode, to the [application window](output_wnd.htm), press the Esc key to return to the editor or debug window. Also available with the F4 function key shortcut.

### <span id="status line">Move to status line</span>

Move the cursor into the command combobox on the toolbar. This is also available with F2. Once in the box the names of objects may be entered directly and actioned by pressing RETURN.

### <span id="Large variables">Large variable display</span>

Displays an ASCII and hexadecimal dump of the last actioned alpha variable. Not currently implemented.

### <span id="List return">List return</span>

Displays the current subroutine return stack as a dialog providing a back trace. For details see [Back Trace Dialog](BackTrace_Dialog.htm).

### <span id="List dim">List dim</span>

Displays all variables. See [List Dim Dialog](List_Dim_Dialog.htm).

### <span id="current variables">Display current variables</span>

Obsolete, to be removed.

### <span id="revert">Revert to default</span>

This restores the window layout to the default for that Scene tiling them to fill all the available space in the parent workbench window.

### <span id="tidy">Tidy</span>

This minimizes all the windows except the one with focus.

### <span id="cascade">Cascade</span>

This lays out all the windows one on top of another slightly offset in a cascade string in the top left. This allows the window titles to be read and can be useful for finding a window otherwise obscured.
