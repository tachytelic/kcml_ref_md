## <img src="bitmaps/KEYBRD.png" data-border="0" width="32" height="32" alt="keyboard icon" /> Keyboard shortcut keys

These keys, when pressed in combination with CTRL, are active when editing program text or debugging.

| Key | CTRL | SHIFT+CTRL |
|----|----|----|
| A | Save as ... |  |
| B | Match brackets | Extend selection to matching bracket |
| C | Copy |  |
| D | [LIST DIM dialog](List_Dim_Dialog.htm) |  |
| E | Find next object definition, e.g. DEFFORM | Find previous object definition |
| F | [Search](Searching_for_text.htm) | KCML 5.02 object browser (now obsolete) |
| G | Show information about program line at cursor | CLEAR P and load library program containing the line at the cursor into foreground |
| H |  |  |
| I | Toggle [collapse mode](Collapse_Mode.htm) |  |
| J |  |  |
| K | Start/stop selection |  |
| L | Add string to chevron at current language position |  |
| M |  |  |
| N | Find Next |  |
| O | Open program ... |  |
| P | Find Previous |  |
| Q | CVS strip |  |
| R | Collapse multiline REMs |  |
| S | [Save program](saveprog.htm) |  |
| T | LIST RETURN [trace dialog](BackTrace_Dialog.htm) |  |
| U | Undo | Redo |
| V | Paste |  |
| W | CVS match | CVS match select |
| X | Cut |  |
| Y | Delete current line |  |
| Z | Make current line the top of the editor/debugger window. Press again to put the line at the center of the window. |  |
| Break | HALT |  |

The RESET key is CTRL-ALT-break.
