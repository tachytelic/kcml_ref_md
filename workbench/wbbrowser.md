## The Browser Window

The browser window allows the components of a program to be displayed in a structured way allowing complex programs to be navigated. It supports five modes selectable with the first five toolbar buttons on the special browser toolbar. The last down arrow button will bring up a menu specific to the currently selected mode. Currently it is only active for forms and files.

<img src="bitmaps/browsebuttons.png" data-border="0" usemap="#browsebuttons" alt="Browser toolbar in forms browsing mode" />

|  |  |
|----|----|
| [Forms](wbbrowseforms.htm) | Displays the forms in the program drilling down to their component controls, methods, properties and events. |
| [Objects](wbbrowseobjects.htm) | Lists the [COM](mk:@MSITStore:kcmlrefman.chm::/comintro.htm), [OCX](mk:@MSITStore:kcmlforms.chm::/IntroOCX.htm), or [SOAP](mk:@MSITStore:kcmlforms.chm::/ObjSoap.htm) objects in a program expanding out to show their methods, properties and enumerations. It also provides the ability to store references to these objects. |
| [Functions](wbbrowsesubs.htm) | Shows all the subroutines in a program in sorted order. |
| [Database](wbbrowsedb.htm) | Allows you to browse information about the databases defined in your **kconf.xml**. |
| [Files](wbbrowsefiles.htm) | Displays a file tree allowing programs to be loaded. |
