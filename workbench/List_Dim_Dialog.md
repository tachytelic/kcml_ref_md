## LIST DIM dialog

This dialog displays variables currently in scope in the order of declaration on the variable chain. The Variable edit box can be used to enter a search mask to be selective. The metacharacters ? and \* are recognized as matching one character or a number of characters respectively. To see all the variables use \*. In the example below all the variables on the chain that start with GB_D are selected with a mask of GB_D\*. The Variable field is case-insensitive.

The first column will be a C for a common variable, a D for a non-common variable or L for a local variable. The second column is the name of the variable with any appropriate dimensioning information. The third column is the current value. String variables may be truncated to fit.
