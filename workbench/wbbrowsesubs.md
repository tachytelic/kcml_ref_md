## Function Browser

<img src="bitmaps/browsemods.png" data-border="0" width="183" height="102" alt="Browsing modules" />

This uses a tree control to list the components of the program, the foreground, any SELECT@PART global and any modules loaded. The last folder displays [KCML Constants](mk:@MSITStore:kcmlrefman.chm::/tmp/constindex.htm) which are non-user defined KCML language constants. If the Global/Module is protected then it is possible to set a password for it using this control. For a protected module you can use the context menu to display the password dialog.

Within each of these components the tree can be expanded to show in sorted order all the subroutines defined in that component. By clicking on an entry it can display the function definition in the main editor window. This can be done in both edit and debug modes. A context menu option is also available. The context menu provides the further option of being able to place a trap on the function. The tree will also show any record structures defined in each component.
