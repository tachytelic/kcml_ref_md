The mouse can be used in a point and click fashion to examine variables and display subroutines as well as marking out text for cut and paste editing.

The following functionality is available.

| Shift key state | Mouse action | Function |
|----|----|----|
|  | Left button down | Move cursor to mouse pointer |
|  | Left button drag | Mark section of text |
|  | Left button up | Stop marking section of text |
|  | Left double click | Select object under mouse pointer |
|  | Right double click | Invoke a [context menu](wbcontext.htm) |
| Shift | Left button down | Make end of marked section at mouse pointer |
| Shift | Left button drag | Make end of marked section at mouse pointer |
| Ctrl | Left button down | Move cursor to mouse pointer |
| Ctrl | Left button drag | Mark section of text. Marking extends so that whole statements are always included |
| Ctrl | Left double click | Select statement under mouse pointer |
| Ctrl-Shift | Left button down | Mark end of marked section at mouse pointer, selecting whole statement |

When marking text by dragging, the screen can be forced to scroll automatically by dragging the mouse pointer off the top or bottom of the screen. Whilst the mouse is outside the screen area scrolling repeats automatically.
