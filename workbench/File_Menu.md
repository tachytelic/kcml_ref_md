# File Menu

This menu allows the loading and saving of programs. It contains a list of up to the last 9 programs loaded allowing a program in the list to be reloaded directly. Finally it also allows you to exit KCML.

<img src="bitmaps/filemenu.png" data-align="TOP" data-border="0" alt="File menu" />

### New

<span id="new"></span>

This will clear out the current program and allow a new program to be edited.

### Open ...

Used to open a KCML program file. Once selected a dialog box is displayed allowing the working directory and program file to be selected. Programs can also be loaded from the [file browser](wbbrowsefiles.htm).

### Save

<span id="save"></span>

Used to save the program or program overlays in memory under their original names. Once selected a dialog box is displayed suggesting how this should be done. See [saving programs](saveprog.htm) for more information.

### Save As ...

Used to save a KCML program file under a different name. Once selected a dialog box is displayed allowing the working directory and program file to be selected.

### Print

Allows a code listing to be printed. This requires a KPrint Server inorder to function. See [printing programs](printprog.htm) from more information.

### Info

Displays some information about the loaded program on the status bar.

### Exit

Terminates KCML and the Workbench. If the program in memory has been changed you will get a dialog box warning you that the modifications have been last and giving the opportunity to cancel the operation.
