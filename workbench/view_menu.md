## View Menu

These menu options allow the configuration of the toolbar and the MDI windows. Not all the available windows are visible at all times and this menu, which enumerates them all, can be used to bring a hidden window to the front.

<img src="bitmaps/viewmenu.png" data-align="BASELINE" data-border="0" alt="View menu" />

### Toolbar

<span id="toolbar_menu"></span>

Brings up a child menu allowing selection of the various optional [toolbar](toolbar.htm) sections to be shown.

<img src="bitmaps/toolmenu.png" data-border="0" data-align="left" alt="Configure visible toolbars" />

|  |  |
|----|----|
| [File](toolbar.htm#file) | File load and save options |
| [Clip](toolbar.htm#clip) | Clipboard options |
| [Edit](toolbar.htm#edit) | Search and replace and object options |
| [Debug](toolbar.htm#debug) | Debugger options (available only in debugger) |
| [View](toolbar.htm#view) | Buttons to enable the component windows as controlled by the [view menu](view_menu.htm) |
| [Scene](toolbar.htm#scene) | Buttons for the available [scenes](scene.htm) |
| [FnBar](toolbar.htm#fnbar) | A combo box for entering expressions |
