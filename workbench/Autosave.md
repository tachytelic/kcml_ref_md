## Autosave settings

The Workbench can be instructed to save the current program to disk regularly as a precautionary measure. By default no autosave occurs. This dialog allows it to be enabled and the time between saves to be defined. You can also set the filename to be used. After a crash this file can be LOADed to restore the workbench environment.
