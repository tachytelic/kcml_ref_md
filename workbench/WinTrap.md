## Trap window

This window displays any traps currently defined. Traps can be set by clicking on the left margin of a line in the editor, with a right click [context menu](wbcontext.htm) or with the immediate mode [TRAP](mk:@MSITStore:kcmlrefman.chm::/TRAP.htm) command in the [console window](console_wnd.htm). To refresh the display after setting the trap press F5.

Traps can be conditional and can be used for tracing. You can edit the traps conditions by clicking on a trap in this window. This brings up the [trap dialog](trap.htm). A right click context menu also allows editing as well as deleting a selected trap.
