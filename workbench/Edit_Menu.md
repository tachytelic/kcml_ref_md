## Edit Menu

These options are used in cut and paste editing.

<img src="bitmaps/editmenu.png" data-align="TOP" data-border="0" usemap="#editmenu" width="214" height="212" alt="Edit menu" />

### <span id="undo">Undo</span>

Used to undo previous program modifications. Up to 100 previous modifications can be undone. You can also use the shortcut keys CTRL-U or CTRL-BACKSPACE.

### <span id="redo">Redo</span>

Used to redo previous undo commands. Like undo up to 100 previous redoes can be performed. You can also use the shortcut keu SHIFT-CTRL-U

### <span id="cut">Cut</span>

Cuts any currently marked text into the clipboard. You can also use the shortcut key CTRL-X or SHIFT-DELETE.

### <span id="copy">Copy</span>

Copies any currently marked text into the clipboard allowing it to be pasted either in the Workbench or in another Windows application. You can also use the CTRL-C or CTRL-INSERT shortcut keys.

### <span id="paste">Paste</span>

Insert the contents of the clipboard at the current cursor position. You can also use the CTRL-V or SHIFT-INSERT shortcut key.

### <span id="show paste">Show paste buffer</span>

Displays the contents of the paste buffer in the editor window. Press any key to redisplay the editor.

### <span id="cvs match">CVS match</span>

Toggles between the alternatives for the current CVS conflict. You can also use the CTRL-W shortcut key. See [source control in KCML](cvs.htm) for more information.

### <span id="cvs match select">CVS match select</span>

Selects the whole of the conflicting alternative that contains the cursor. You can also use the SHIFT-CTRL-W shortcut key. See [source control in KCML](cvs.htm) for more information.

### <span id="cvs strip">CVS Strip</span>

This will remove the selected lines between the CVS markers that contain the cursor. You can also use the CTRL-Q shortcut key. See [source control in KCML](cvs.htm) for more information.
