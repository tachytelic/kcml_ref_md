## Console window

The Console Window can be called up or sent to the background in the editor by using the TAB key. It provides a colon prompt, immediate mode and access to the old [line oriented editor](LineEditor.htm) from KCML 3.2. It is also possible to debug using [commands](mk:@MSITStore:kcmlrefman.chm::/consoledebug.htm). The window is re-sizable.

Editing in this window is line oriented using the keystrokes of the old [editor](LineEditor.htm) and imposes the same line length restrictions of 1900 characters per line.
