## Object Menu

These options are used with forms and other objects

<img src="bitmaps/objmenu.png" data-align="BASELINE" data-border="0" alt="Object menu" />

### Object browser

This refers to the now obsolete object browser from KCML 5.02. It is supported for backward compatibility only.

<span id="expand"></span>

### Expand all

Not available in debug mode. This will expand all the [DEFEVENT](mk:@MSITStore:relnotes.chm::/DEFEVENT.htm) event handlers in the program revealing their statements. This can be done for a particular form or DEFEVENT by clicking on the [DEFFORM](mk:@MSITStore:relnotes.chm::/DEFFORM.htm) or DEFEVENT.

<span id="collapse"></span>

### Collapse all

Not available in debug mode. This will collapse all the [DEFFORM](mk:@MSITStore:relnotes.chm::/DEFFORM.htm) statments in the program hiding the statements in their event handlers, though still showing the [DEFEVENT](mk:@MSITStore:relnotes.chm::/DEFEVENT.htm) statments themselves, thus making the layout of the program more obvious. This can be done for a particular form or DEFEVENT by clicking on the DEFFORM or DEFEVENT toggling the leading - sign to a + sign.

<span id="nextobj"></span>

### Find next object definition

This will search for the next [DEFFORM](mk:@MSITStore:relnotes.chm::/DEFFORM.htm) or [DEFOBJ](mk:@MSITStore:relnotes.chm::/DEFOBJ.htm) statement starting from the current cursor position. This has a keyboard shortcut of CTRL-E

<span id="prevobj"></span>

### Find previous object definition

This will search backwards for the previous DEFFORM or DEFOBJ statement starting from the current cursor position. This has a keyboard shortcut of SHIFT-CTRL-E
