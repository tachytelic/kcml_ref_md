## Function Menu

<img src="bitmaps/FnContext.png" data-border="0" data-align="MIDDLE" alt="Context menu for functions" />\
\

The default in this case is **Definition** so you can click again or press RETURN to get the same effect as the old right double click.

The options on the menu are

|  |  |
|----|----|
| Definition | Will move the editor window view to the location of the function definition. |
| Help | Will display the help for this function. The help is gathered from adjacent REM statements. |
| Revisit | This will move the editor view back to the previous entry in the visit chain. Example: if you use the definition option for this menu to jump off to a function definition you can use revisit to jump back to where you came from. |
