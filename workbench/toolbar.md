## The toolbar

The toolbar contains a number of buttons divided into a number of optional sections. The Toolbar submenu on the [View menu](view_menu.htm#toolbar_menu) allows you to switch on or off the sections you require. Some of the sections are modal and are greyed out in edit or debug modes. The buttons are tooltipped so their function can be discovered by hovering the mouse over the unknown button.

The sections are listed below. Hover the mouse over any button to see more about that option.
