## Print Dialog

This allows you to print a code listing. The code can only be printed via a KPrint Server. The KPrint server should also have the KCML Listing forms for best results.

<img src="bitmaps/PrintDialog.png" data-border="0" usemap="#PrintDialog" width="353" height="136" />

The combobox is used to specify the KPrint queue and server in the form **queue@server:port**. This holds a list of recently used KPrint servers, to view use the drop-down arrow to the right of the combo box.

Its possible to set the layout of the printed document. The two options available are landscape and portrait. Landscape is best for heavily indented code although you will not get as many lines per page as portrait.

A choice between printing the whole code listing or only the selected text can also be made. The option to print line numbers can only be selected in the 'Complete Listing' mode.
